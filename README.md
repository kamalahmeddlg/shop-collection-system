# 💰 Shop Money Collection System

A simple and clean money collection system built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add customer collection
- View collection history
- Delete entries
- Total collection dashboard
- LocalStorage data saving

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Future Improvements
- Login system
- Database integration (PHP + MySQL)
- Receipt generation
- Admin panel
